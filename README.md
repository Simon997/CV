## 1. New project

    $ git init

## 2. add to tree

    $ git add .

## 3. Snapshot

    $ git commit --m "<message>"

## 4. add the cloud!

    $ git remote add origin git@github.com:<USERNAME>/<REPO>.git

## 5. change the branch

    $ git checkout -b gh-pages

## 7. push to GitHub

    $ git push origin gh-pages
